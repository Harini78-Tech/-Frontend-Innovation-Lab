
# Smart Expense Tracker 

## Setup

1. Ensure you have Node.js and npm installed.
2. Ensure MySQL is installed. This project uses port **3306** by default (you can change it in `backend/.env`).
3. From the project root:
   ```bash
   npm install
   npm run dev
   ```
4. Open http://localhost:8000

## Notes
- Backend files are in `backend/`.
- Frontend files are in `frontend/`.
- Database will be created automatically if missing.
